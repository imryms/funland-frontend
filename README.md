### ***Description***
#### FunLand is a Full-stack booking platform that allows users to instantly reserve park tickets.
***

### ***Technologies Used***
* Frontend: React.js & Axios.
* Backend: Node.js & Express.
* Database: MongoDB & Mongoose.
* Styling: CSS
